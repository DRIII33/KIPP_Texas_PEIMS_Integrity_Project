# KIPP_Texas_PEIMS_Integrity_Project
Full-stack data pipeline (Python, SQL, BigQuery) for KIPP Texas Public Schools. Proactively audits PEIMS/SAAH compliance and FSP funding risk (HB 2, SPED services). Simulates a State Reporting Data Specialist role, delivering actionable Looker Studio insights for Regional Support and Leadership to mitigate audit exposure.
